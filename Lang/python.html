Pengenalan Python

Python adalah bahasa pemrograman tingkat tinggi yang diciptakan oleh Guido van Rossum. Python dikenal dengan sintaksnya yang sederhana dan mudah dibaca, membuatnya menjadi pilihan ideal untuk pemula.

Keunggulan Python

  • Mudah dipelajari dan digunakan
  • Sintaks yang bersih dan readable
  • Multi-paradigm programming
  • Library yang sangat lengkap
  • Community yang besar dan aktif

Penggunaan Python

  • Web Development (Django, Flask)
  • Data Science & AI
  • Automation & Scripting
  • Game Development
  • Desktop Applications

Sintaks Dasar Python

Python memiliki sintaks yang bersih dan mudah dibaca. Indentasi adalah bagian penting dari sintaks Python untuk mendefinisikan blok kode.

Aturan Indentasi

  • Gunakan 4 spasi untuk setiap level indentasi
  • Jangan campur spasi dan tab
  • Indentasi yang salah akan menyebabkan error

Contoh Sintaks Dasar

# Variabel dan tipe data
nama = "John"          # String
umur = 25              # Integer
tinggi = 175.5         # Float
is_student = True      # Boolean

# Print dengan format
print(f"Nama: {nama}")
print(f"Umur: {umur} tahun")
print(f"Tinggi: {tinggi} cm")
print(f"Status: {'Mahasiswa' if is_student else 'Bukan Mahasiswa'}")

# Komentar multi-line
"""
Ini adalah komentar
yang berjalan di beberapa baris
"""

Input dari User 

# Menerima input dari user
nama = input("Masukkan nama Anda: ")
umur = int(input("Masukkan umur Anda: "))

print(f"Halo {nama}, umur Anda {umur} tahun.")

Tipe Data di Python

Python memiliki beberapa tipe data built-in yang penting untuk diketahui. Setiap tipe data memiliki karakteristik dan penggunaan yang berbeda.

Tipe Data Numerik

  • int - Bilangan bulat
  • float - Bilangan desimal
  • complex - Bilangan kompleks

Tipe Data Sequence

  • str - String
  • list - List
  • tuple - Tuple

Contoh Penggunaan Tipe Data

# Tipe Data Numerik
x = 10          # int
y = 3.14        # float
z = 2 + 3j      # complex

print(type(x))  # 
print(type(y))  # 
print(type(z))  # 

# Tipe Data String
nama = "Python"
kalimat = 'Saya belajar Python'
multiline = """Ini adalah
string multi-line"""

print(nama.upper())      # PYTHON
print(nama.lower())      # python
print(len(kalimat))     # 17

# Type Conversion
num_str = "123"
num_int = int(num_str)
print(num_int + 10)      # 133

Struktur Kontrol

Struktur kontrol memungkinkan program untuk membuat keputusan dan mengulang eksekusi kode berdasarkan kondisi tertentu.

Conditional Statements (If-Else) 

# If statement sederhana
umur = 18

if umur >= 18:
    print("Anda sudah dewasa")
    
# If-Else
nilai = 75
if nilai >= 80:
    print("Nilai A")
elif nilai >= 70:
    print("Nilai B")
elif nilai >= 60:
    print("Nilai C")
else:
    print("Nilai D")

# Nested If
x = 10
y = 20
if x > 5:
    if y > 15:
        print("Kedua kondisi terpenuhi")

Loops (Perulangan) 

# For loop
print("For loop:")
for i in range(5):
    print(f"Iterasi {i}")

# For loop dengan list
buah = ["apel", "jeruk", "mangga"]
for item in buah:
    print(f"Saya suka {item}")

# While loop
print("\nWhile loop:")
count = 0
while count < 3:
    print(f"Count: {count}")
    count += 1

# Loop dengan break dan continue
print("\nLoop dengan break:")
for i in range(10):
    if i == 5:
        break
    print(i)

print("\nLoop dengan continue:")
for i in range(10):
    if i % 2 == 0:
        continue
    print(i)
